# workflow-automator

A workflow automation app, where each stage is a block. Each stage represents an LLM computation or a file/input text from the user.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/workflow-automator.git
cd workflow-automator
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
